# Patronum 撰寫風格鐵則

適用範圍：所有文章本文、persona 文案（site.ts／首頁／about）、由我（Claude Code 帳號）或 subagent 撰寫的一切內容。引擎 `engine/write` 的 SYSTEM prompt 也須內含本鐵則。

## 硬性禁止（100% 遵守）

1. **破折號（em dash `—`／`——`）一律禁用。** 改用句號、逗號或分開兩句。
2. **禁止排比句式。** 不要連續三個結構相同的短句／詞組堆疊（例：「有人哭著，有人笑著，有人走了」）。
3. **禁止對比句式。** 不要「不是…而是…」這種對照框架。
4. **禁止油膩三段式。**（鋪陳→轉折→昇華 的那種三拍收尾。）
5. **禁止過度正面／雞湯化的描述詞與口頭禪**，包含但不限於：
   - 「穩」「撐」「懂」
   - 「不是…而是…」
   - 「其實」
   - 「很簡單」

## 必須

- **繁體中文、台灣用語**（用「老人家／長輩」「喜事」「家人」這類台灣慣用語，不用對岸詞彙）。
- 句子平實、像人講話，不堆修辭。
- persona 第一人稱「我」，有情感但不煽情。
- 後台紀律（呈現光譜／不裁決／溫柔／AI 揭露）放在 editorial-policy 與 disclosure，**不可當成 persona 的台詞**。

## 自我檢查（送出前 grep）

- `grep -n "—"` 必須 0 筆。
- 通讀一遍：有沒有三連排比？有沒有「不是…而是」？有沒有「其實／穩／撐／懂／很簡單」？有就改掉。
